# Catalog RFQ Admin

React administration UI for the catalog and quote-request API.

It includes the operational dashboard, products, categories, brands, attributes, filters, media, quote-request details and workflow, contacts, notifications, team roles and invitations, reports, and audit log.

```bash
npm install
npm run dev
npm run typecheck
npm run lint
npm run build
```

Set `VITE_API_BASE_URL=/api` when the API is reverse-proxied on the same host.
